基于 STM32F405RGT6 的触控无线开发板

一块面向真实产品形态演示与快速原型打造的开发板。集成触摸交互、LoRa 通讯、Wi-Fi 联网与 LVGL 图形界面，帮助开发者把想法更快落地为可交互设备。

Product Overview

Cipher Node Touch 特点

高性能 MCU

基于 STM32F405RGT6，主频 168MHz，兼顾实时控制、通信处理与图形交互。

WiFi 联网

支持WiFi，助力万物互联与智能应用。

LoRa 通讯能力

支持 LoRa 文本消息与链路交互，适用于远距离低功耗通信演示与应用验证。

触摸屏支持

面向交互设备开发，适配多种触控屏方案，快速搭建人机界面。

LVGL 图形界面

成熟 GUI 方案，便于构建流畅、可定制、易维护的视觉界面。

AI 辅助

项目目录内置 SKILL.md，适合 AI agent 协助开发，助力学习AI在嵌入式开发中的应用。

传感器

板载高性能温湿度传感器。

电池扩展支持

预留锂电池充电电路与接口，用户可自行增加锂电池，可嵌入壳体内部。电池尺寸不大于 70mm × 40mm，厚度不大于 5mm。

USB 通信

支持 USB 通信，支持U盘模拟与虚拟串口，支持U盘固件升级。

开源生态

文档、示例、驱动与工具链持续开放，便于二次开发与社区协作。

双编译环境支持

Demo 同时支持 CMake 与 Keil 两种编译环境，适配不同团队的开发习惯与工程流程。
